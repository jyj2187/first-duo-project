## 기능 구현 우선순위

| 기능       | 설명            | 우선순위 |
|----------|---------------|------|
| Tag      | 다대다 매핑        | 1순위  |
| 검색기능     | Repository이용  | 2순위  |
| 회원프로필    | 템플릿생성, 사진업로드  | 1순위  |
| 댓글       | 일대다           | 2순위  |
| 파일업로드    | 글에 파일 업로드 가능  | 1순위  |
| 좋아요      | 글에 좋아요(추천) 가능 | 2순위  |
| 카테고리     | 글 종류 분류       | 1순위  |
| 페이지네이션   | 페이지 분석        | 1순위  |
| API 컨트롤러 | JSON 형식 응답    | 2순위  |
